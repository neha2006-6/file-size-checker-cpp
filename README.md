#  File Size Checker in C++

A simple and useful C++ utility that checks 
the size of any file on your computer!

## Features
-  Shows file size in Bytes, KB, MB or GB
-  Checks if file exists
-  Clean and simple output
-  Works on any file type

## How to Run
```bash
g++ file_size_checker.cpp -o file_size_checker
./file_size_checker
```

## Example Output
Size Checker in C++
Enter filename: hello.txt
✅ =File found!
📁 Filename : hello.txt
📏 Size     : 2.5 KB
📊 Bytes    : 2560 bytes

## 🛠️ Technologies Used
- C++
- File Handling (fstream)
- Standard Template Library (STL)

## Requirements
- C++ compiler (g++)
- Any operating system (Windows, Linux, Mac)
